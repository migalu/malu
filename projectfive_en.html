ACDSee Design

2.Semester, teamwork

Description

ACDSee Photo Manager for Mac is a free software for image preview in multiple formats, such as BMP, JPEG, PNG and TIFF files. User can simply edit photos, and sort photos.

Problem

The information structure of current ACDSee Photo Manager for Mac is not very clear but with complicate manipulations. Besides, the style of UI is tedious with blurred meaning on icons, which causes increments of study costs. Hence, we are stressing on strengthen the core function, eliminate unnecessary points and add the function of sharing corresponding to customer habit. In addition, we have also modified operating procedures in order to derive more frequent and fluent use. Our three main point on design: 1. Optimization on photo array and view pattern; 2. Rapid edit on photo; 3.Simplification on arranging photos.
